Class FS_Api

Class FS_Api

Wraps Freemius API SDK to handle: 1. Clock sync. 2. Fallback to HTTP when HTTPS fails. 3. Adds caching layer to GET requests. 4. Adds consistency for failed requests by using last cached version.
